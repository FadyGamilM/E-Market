To Run the application =>
 
		1) Download the zip file
		2) Extract zip file application folder
		
	We need to intall the server side first =>
		1) Open api folder 
		2) Open the terminal in this current location path (api folder)
		3) After that we need to install node modules by write in the terminal => npm i 
		4) To run server side write in the terminal => npm run server 
		
	After that we need to install the client side =>
	
		1) Open client folder 
		2) Open the terminal in this current location path (client folder)
		3) After that we need to install node modules by write in the terminal => npm i 
		4) To run client side write in the terminal => npm start
